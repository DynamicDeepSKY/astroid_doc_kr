시스템 업데이트

시스템 버전 확인

시스템 설정으로 이동하여 “System info”를 클릭해 시스템 정보 창을 엽니다. 시스템 버전이 홈페이지에 있는 버전보다 오래되었다면, 시스템을 업데이트할 수 있습니다.

온라인 시스템 업데이트

  1. 이더넷 케이블 또는 외부 WIFI를 통해 장치를 인터넷에 연결합니다. 시스템 설정 메뉴에서 이메일 또는 전화번호를 등록했다면, 장치의 IP 주소를 메시지로 받게 됩니다.

  2. 웹 인터페이스를 열고 설정 메뉴 -> “System Setting”으로 이동하여 “Check system update”를 클릭합니다. 새 버전이 있을 경우, 시스템 업데이트가 자동으로 시작되며 상단에 USB 아이콘 이 표시됩니다. 시스템 로그에서 진행 상황을 확인할 수 있습니다. 시스템 로그를 보려면 “System Setting” 메뉴 아래의 “Show System Log”를 클릭하십시오.

  1. 업데이트 아이콘 이 사라질 때까지 기다립니다.

  2. 시스템을 재부팅합니다.

  3. 시스템 버전이 최신인지 확인합니다.

오프라인 시스템 업데이트

  1. DynamicDeepSKY.com에서 업데이트 파일을 다운로드합니다. 업데이트 파일은 지원 탭 아래에 있습니다.

  2. 다운로드한 파일을 DDS USB에 복사합니다. DDS USB가 없다면, 자세한 정보는 Making a DDS USB 를 참조하십시오.

  3. USB를 Astroid에 삽입합니다. 상단에 USB 아이콘 이 표시됩니다.

  4. 업데이트 아이콘 이 사라질 때까지 기다립니다.

  5. 시스템을 재부팅합니다.

  6. 시스템 버전이 최신인지 확인합니다.
